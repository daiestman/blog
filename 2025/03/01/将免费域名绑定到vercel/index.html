将免费域名绑定到vercel

2025-03-01
未分类
1.2k 词
  • 网络

有了免费的服务器，一个免费的域名必不可少，接下来我们将免费域名us.kg域名绑定到vercel。

us.kg 注册并托管到 Vercel 详细教程

一、注册 us.kg 域名

  1. 访问官网：打开 us.kg 官网，点击页面上的“Register”按钮进入注册页面。

  2. 填写注册信息：

    • 用户名：填写您自定义的用户名，可以是您的用户名或项目名称。
    • 邮箱：输入您的常用邮箱地址，用于接收验证邮件。
    • 电话与地址：us.kg 域名注册要求提供联系信息，可以使用虚拟信息，如通过“美国地址生成器”生成美国地址。

  3. 验证邮箱：注册成功后，登录您的邮箱，找到 us.kg 发送的验证邮件，点击邮件中的验证链接完成邮箱验证。

  4. 完成 KYC 验证：

    • 登录 us.kg 后台，点击“Verify your account”。
    • 选择“GitHub KYC Verification”选项。
    • 按照提示填写相关信息，如用户名、用途等。
    • 提交 Issue，然后勾选“我已提交 Issue”选项。
    • 给项目点 star，等待审核通过。

  5. 申请域名：

    • 登录 us.kg 后台：US.KG Domain Registry and Management Authority。
    • 输入想要的域名前缀，点击“Check”按钮。
    • 若显示域名可用，先不填写 Name Server（域名服务器），保持页面不关闭。

二、托管域名到 Vercel

  1. 注册并登录 Vercel

    • 访问 Vercel 官网，点击“Sign Up”按钮。
    • 选择使用 GitHub、GitLab 或 Bitbucket 账号登录，建议使用 GitHub 账号，因为大多数项目代码托管在 GitHub 上。
    • 按照提示完成注册和登录过程。

  2. 创建新项目

    • 登录 Vercel 后台，点击“New Project”按钮。
    • 选择代码所在的 Git 仓库（如 GitHub 仓库）。
    • Vercel 会自动检测项目设置，包括构建命令、输出目录等。
    • 可以根据项目需求修改这些设置，如构建命令可能是 npm run build 或 yarn build。

  3. 部署项目

    • 点击“Deploy”按钮，Vercel 会自动拉取代码并进行部署。
    • 部署过程中可以看到详细的构建日志。
    • 部署成功后，Vercel 会提供一个默认的域名，格式为 your-project-name.vercel.app。

  4. 配置自定义域名

    • 点击项目名称，进入项目详情页面。
    • 在左侧菜单中选择“Domains”选项。
    • 点击“Add Domain”按钮，输入您申请的 us.kg 域名。
    • Vercel 会提示您添加 DNS 记录，将域名的 DNS 记录指向 Vercel 提供的地址。

三、注意事项

  1. 域名续期：us.kg 域名是完全免费的，使用期限为 1 年，到期前 180 天可以在控制面板申请续期。
  2. Vercel 部署：确保您的项目代码已托管到 GitHub 或其他 Git 平台，并且可以正常构建和运行。
  3. 域名解析：在配置自定义域名时，确保域名的 DNS 记录正确指向 Vercel 提供的地址，否则网站可能无法正常访问。

通过以上步骤，您可以成功注册 us.kg 域名并将其托管到 Vercel 上，享受免费的域名和网站托管服务。

加速工具（可选）

  • 电脑端：使用 Watt Toolkit (Steam++) 加速 GitHub/Vercel
  • 手机端：自行搜索合法加速工具（注意安全风险）

创建仓库

  1. 登录 GitHub，点击右上角 + → New repository
  2. 填写仓库信息：
    • Repository name: 任意名称（如 my-hexo-blog）
    • Visibility: 选择 Public
    • Initialize this repository: 不勾选

步骤 5：通过 Vercel 加速访问

  1. 访问 Vercel，用 GitHub 账号登录
  2. 导入仓库：
    • 点击 Add New Project → 选择你的博客仓库
    • 直接点击 Deploy（无需修改配置）
  3. 部署成功后，访问 项目名.vercel.app 查看网站

常见问题

部署失败

  • 检查 _config.yml 的缩进（必须为空格）
  • 确保 GitHub 仓库链接正确

访问速度慢

  • Vercel 默认海外节点，可尝试绑定国内 CDN
  • 或使用 Coding Pages 替代

加速工具

  • 电脑端：Watt Toolkit (Steam++)
  • 手机端：前往404应用商店下载
